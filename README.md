# Transaction Fraud Detection Using Machine Learning

## 📌 Project Description
This project detects fraudulent financial transactions using machine learning techniques.
The system classifies transactions as fraudulent or legitimate based on transaction and identity features.

## 📊 Dataset
- **Name:** IEEE-CIS Fraud Detection Dataset
- **Source:** Kaggle (IEEE Computational Intelligence Society)
- Dataset files are not uploaded due to size and license restrictions.

## 🧠 Machine Learning Models
- Logistic Regression
- Random Forest Classifier

## ⚙️ Techniques Used
- Data preprocessing & cleaning
- Label Encoding
- Feature Scaling
- Stratified K-Fold Cross Validation
- Threshold tuning
- Confusion Matrix & Feature Importance

## 📈 Results
- ROC-AUC Score: ~0.89
- Accuracy: ~97%
- Random Forest performed better than Logistic Regression
